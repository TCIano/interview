# 基本数组方法
#### Array.isArray（）
监测参数里面的值是否为一个数组，是返回 true，不是返回 false
#### **** instaceof Array
检测一个数据是否为一个数组，是返回 true ，不是返回 false
#### 数组增删 
push（*，*，...）
将一个或者多个元素添加到数组末尾，返回数组长度
pop（）
将数组中的最后一个元素删除，返回该元素的值，空数组调用pop（），会返回undefined
unshift（*，*，...）
将一个或者多个元素加到数组开头，返回数组长度
shift（）
将数组中第一个元素删除，返回该元素的值，空数组调用shift（），会返回undefined

#### 数组查找
indexOf（查找元素，开始查找的位置索引）
从前往后返回首个被找到的元素的索引，没有返回 -1
lastIndexOf（查找元素，开始查找位置的索引）
从后往前查找，返回最后一个满足条件元素索引，没有返回-1
includes（要搜索的元素，开始搜索的位置（默认为0））
确定数组是否包含某个元素，包含返回 true ，不包含返回 false
find（（数组中当前项，当前项索引，整个数组）=> { }）
返回数组中满足回调函数条件的第一个元素，找不到会返回 undefined
findLast（（数组中当前项，当前项索引，整个数组）=> { }）
返回数组中满足回调函数条件的最后一个元素，找不到会返回 undefined
findIndex（（数组中当前项，当前项索引，整个数组）=> { }）
返回数组中满足回调函数条件的第一个元素的索引，找不到会返回 -1
findLastIndex（（数组中当前项，当前项索引，整个数组）=> { }）
返回数组中满足回调函数条件的最后一个的索引，找不到会返回 -1
#### 数组转为字符串

1. join（ ’分隔符‘）

一个数组所有元素连接成一个字符串并且返回这个字符串，长度为 0 返回空字符串
不传递参数，默认用逗号分割。

2. toString（）

连接数组里的每一个元素，返回一个用逗号分开的字符串，
#### 其他方法

1. concat（*，*，...）

合并两个或者多个数组，返回一个新数组。

2. slice（begin，end）

begin：可选，若为负数，表示原数组中倒数第几个元素。若省略，则默认从索引0开始。
end：可选，提取原数组从 begin（包含begin） 到 end（不包含end）的所有元素
    省略，一直提取到末尾。

3. splice（开始索引，删除个数，要添加的元素）

# 📝ES5数组新增
#### forEach（function（item，index，arr）{没有return}）
遍历当前数组，不能够更改简单数据类型的值 item值不能改变，但是可以改变 复杂数据类型的值如果arr里面每一项是对象item.属性名 = ***，可以修改。
#### map(function（item，index，arr）{return ***})
对每个数组里面的元素执行函数，返回新数组
新数组里面的值由return的条件决定。
#### filter（function（item，index，arr）{return 表达式}）
筛选数组里满足条件的元素，返回新的数组
return 0 ：return false 表示没满足的元素，返回空数组。
return 1 ： return true 表示元素都满足条件，返回原来数组
return 表达式 ：返回满足表达式的元素生成的数组
#### every（function（item，index，arr）{return 表达式}）
所有满足表达式就返回true
遇到一个元素不满足就返回false
#### some（function（item，index，arr）{return 表达式}）
遇到一个元素满足表达式就返回true
所有元素都不满足 就返回false

# 📝ES6数组新增
#### fill（value，start，end）
用一个固定值来来填充数组从开始索引（包括开始索引）到结束索引（不包括结束索引）的全部元素
value：用来填充数组的值
start：开始索引，默认为 0 ，若为负数，则要+array.length
end：结束索引，默认为 array.length，若为负数，则要+array.length
返回值为改变过的原数组
